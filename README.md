# MemoryMania

A MERN stack application for sharing and managing code snippets.

### Prerequisites
- Node.js (v14 or higher)
- MongoDB (v4.4 or higher)
- npm (v6 or higher)


## Features

- User authentication
- Code snippet sharing
- Syntax highlighting
- Search functionality
- User profiles

## Tech Stack

- Frontend: React + Vite
- Backend: Express.js
- Database: MongoDB
- Authentication: JWT
